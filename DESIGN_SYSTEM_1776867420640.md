# 🎷 FELABRATION FLORIPA — Design System
### Afro-Futurismo Tropical · v1.0 · 2026

---

## 🎨 Princípios de Design

1. **Fela First** — Todo elemento visual deve emanar a energia revolucionária, alegre e indomável de Fela Kuti
2. **Tropical Noir** — Fundos escuros cinematográficos contrastando com cores vibrantes tropicais
3. **Groove Visual** — Micro-animações que dão ritmo à interface, como se o site estivesse dançando
4. **Resistência Elegante** — Sofisticação premium com identidade negra e periférica. Sem diluir. Sem pasteurizar.
5. **Imersão Total** — Cada scroll deve ser uma experiência sensorial completa

| Token | Hex | Uso |

| `--glass-b` | `rgba(255,255,255,0.08)` | Glassmorphism border |

### Gradientes
```css
/* Gold principal */
background: linear-gradient(135deg, #FFD700, #B8960F);

/* Fogo (CTAs de impacto) */
background: linear-gradient(135deg, #E6392B, #B22D23);

/* Logo */
background: linear-gradient(135deg, #FFD700, #E6392B);

/* Timeline */
background: linear-gradient(180deg, #FFD700, #E6392B, #00F0FF);
```

### Tipografia

| Elemento | Fonte | Peso | Tracking |
|---|---|---|---|
| Títulos display (H1) | Bebas Neue | 400 | 2-3px |
| Subtítulos (H2-H3) | Bebas Neue | 400 | 1-2px |
| Body | Inter | 300-500 | 0 |
| Buttons | Inter | 700 | 0 |
| Tags/Labels | Inter | 600-700 | 2-3px |

### Espaçamento
- Section padding: `100px 40px` (desktop), `60px 16px` (mobile)
- Card padding: `20-36px`
- Gap padrão: `24-28px`
- Margin entre seções: `0` (padding cuida)

### Bordas & Sombras
```css
--radius: 16px;      /* Cards, modais */
--radius-sm: 10px;   /* Badges, inputs */
--radius-full: 50px; /* Buttons, pills */

/* Glow dourado (hover) */
box-shadow: 0 20px 60px rgba(255,215,0,0.12);

/* Elevação suave */
box-shadow: 0 8px 32px rgba(0,0,0,0.5);
```

### Motion
```css
--tr: 0.4s cubic-bezier(0.25, 0.46, 0.45, 0.94);

/* Reveal (scroll) */
transition: opacity 0.7s ease, transform 0.7s ease;

/* Hover cards */
transform: translateY(-10px) scale(1.02);

/* Glitch (hero) */
animation: glitch-skew 4s infinite linear alternate-reverse;

/* Pulse (badges) */
animation: pulse 2s infinite;

/* EQ bars (player) */
animation: eq 0.5-0.9s ease infinite alternate;
```

## 🌟 Animações & Micro-Interações

| Elemento | Efeito | Trigger |
|---|---|---|
| Cards de artista | 3D tilt + glow dourado | Mouse hover |
| Botões | translateY(-3px) + shadow expand | Hover |
| Features list | translateX(8px) + border gold | Hover |
| Timeline items | translateX(8px) + border gold | Hover |
| Exp cards | Barra dourada no topo (scaleX) | Hover |
| Gallery items | Zoom 1.1 + overlay fade in | Hover |
| Hero partículas | Canvas particles interativas | Mouse move |
| Countdown | Números com padding animado | Automático |
| Scroll reveal | fadeUp (opacity + translateY) | Scroll |
| Confetti | 80 peças caindo | Click em "Comprar" |
| Player EQ | Barras pulsando | Estado "playing" |
| Glitch text | Micro skew alternado | Contínuo |
